## Decision Rules
- Total Score: 84 points minimum for automatic "GO" decision
- If score is below 84, requires explanation and action plan for proposal submission
- Score below 50% = NO GO

## Decision Categories
1. GO [Green] = Proposal Proceeds
2. GO [Amber] = Proposal Proceeds subject to Action Plan
3. NO GO [Red] = No Proposal

## Approval Requirements
- Score > 70%: Regional Manager approval required
- Score 50-70%: BOD approval required
- Score < 50%: NO GO

## Sign-off Fields
1. Completed By:
   - Name
   - Date

2. Reviewed By:
   - Name
   - Date

## Additional Notes
- All scores must be justified with comments
- Action plan required for Amber status
- Approval timestamps and user tracking required
- Comments/justification required for each scoring category
